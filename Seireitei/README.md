## Why "Seireitei"?

In the anime *Bleach*, the Seireitei is the fortified headquarters of the Soul Society - a place where all important knowledge, governance, and power resides. Just as the Seireitei serves as the central command for Soul Reapers, this directory serves as the central command for all FinSurf project knowledge.

**Alternative name considered:** Daireishokairō (大霊書回廊, Great Spirit Book Corridor) - the massive library within the Seireitei.
