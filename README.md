## 🚀 Quick Commands

```bash
# Development
npm run dev              # Start dev server
npm run build            # Production build
npm run test:backend     # Run Python tests

# Docker
docker compose up        # Local dev stack
docker compose -f deploy/docker-compose.prod.yml up -d  # Production
```

## 🌟 Features at a Glance

- **5 AI Agents** analyzing stocks in parallel (Research, Tax, Dividend, Sentiment, Summary)
- **Real-time timestamps** showing data freshness
- **Recent searches** history for quick re-analysis
- **Professional PDF exports** with HD layout option
- **Tax optimization** insights (short-term vs long-term capital gains)
- **Dividend projections** with Python-calculated accuracy
- **Social sentiment** from Reddit, StockTwits, news, SEC filings
